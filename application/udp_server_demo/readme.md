### 功能描述：
1、支持wifi配网
2、通过menuconfig可配置SSID、PassWord、服务端IP
3、与客户端通信，需要与客户端的IP匹配

### demo编译

```
1、进入SDK目录，在SDK目录下执行指令

2. menuconfig配置参数：通过指令： ./build.sh 芯片型号 应用名称 menuconfig		 举例：./build.sh bl602 udp_server_demo menuconfig
进入配置界面，选择"application",即可查看和配置配网信息。选择"Exit"退出。

3. 编译：通过指令： ./build.sh 芯片型号 应用名称 国家编码 调试等级	举例：./build.sh ln882h udp_server_demo cn debug

4. 编译成功，生成名为："dev_project.bin"的固件，存在于out路径下，根据编译时的平台查找固件，例如：ln882h，则找该平台下对应demo的固件。

5. 固件烧录：不同平台的固件烧录工具不同，应选择正确平台的烧录工具进行固件烧录
```

### demo使用
```
1、该demo是UDP服务端例程。手动复位模组后，串口调试助手查看配网过程信息，注意：为保证下一步执行成功，这一步需要配网成功。

2、电脑连接wifi，注意：这里电脑连接的wifi必须要与模组连接的wifi一致

3、打开网络调试助手，作为客户端，并向服务端发送数据，步骤：
    1）选择协议类型 
    2）输入本地主机地址IP
    3）端口号默认：8888 
    4）点击"打开"即可连接，如果没有连接上，则复位模组或者手动输入IP:端口号，如：192.168.31.21:8888，并重新执行前4步骤
    5）在输入框输入数据，点击“发送”
    注意：模组烧录的是客户端的固件，网络调试助手则作为服务端；反之，模组烧录的是服务端的固件，则网络调试助手则作为客户端

    
4、通过串口调试助手查看服务端是否发送数据和客户端是否收到数据。波特率为：921600。

注意：
    1)、模组与测试电脑必须在同一网段，最简单的方法就是：模组配网的wifi与电脑连接的是同一个wifi。

```
